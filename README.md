## Dev env launch

### Manual

- Install PHP 8.2, Composer, Node LTS, yarn
- Run `composer install`
- Run `yarn install`
- Run `yarn run watch`
- Use php web server to run `public/index.php` file

### Using Docker container

```
docker-compose up --build -d
```
